# Handwritten Digit Recognition

This project implements a machine learning model that recognizes handwritten digits (0–9) using Python and scikit-learn. The model is trained on the built-in digits dataset and predicts the digit from image data.

## Technologies Used
- Python
- scikit-learn
- NumPy
- Matplotlib

## Dataset
The project uses the **digits dataset** provided by scikit-learn.  
It contains images of handwritten digits from 0 to 9 that are used for training and testing the model.

## How the Project Works
1. Load the digits dataset from scikit-learn.
2. Split the dataset into training and testing sets.
3. Train a neural network classifier using the training data.
4. Predict the digit values for the test dataset.
5. Evaluate the model performance using accuracy.

## How to Run the Project

1. Install required libraries
   - pip install scikit-learn matplotlib
   
2. Run the program
   - python digit_recognition.py
   
## Output
The program trains a machine learning model and prints the classification accuracy for handwritten digit recognition.
